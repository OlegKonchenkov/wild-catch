⏰ Timer lato giocatore
Countdown visibile nell'header durante tutta la sessione
A 10 minuti dalla fine → notifica push "⚠️ Mancano 10 minuti!"
A 5 minuti → banner rosso pulsante in-app
Allo scadere → schermata "Evento terminato" con preview classifica
Nessuna nuova azione possibile (incontri bloccati)
🤖 Chiusura automatica
Vercel Cron Job controlla ogni minuto le sessioni attive
Se now() ≥ end_at → status = 'ended'
Trigger automatico: calcola punteggi, genera classifica, aggiorna Hall of Fame
Notifica broadcast ai giocatori via Supabase Realtime
Admin può anche chiudere manualmente in anticipo
