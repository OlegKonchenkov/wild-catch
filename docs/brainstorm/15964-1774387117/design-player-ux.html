5 Player Screens (Bottom Navigation)
🗺️
Mappa
HOME
  • Illustrated Pesaro map (Leaflet + OpenStreetMap)
  • Player position dot (GPS or last known)
  • Encounter radar pulse animation
  • Active mission pin on map
  • QR code scan button
  • Level / EXP / Oro always visible top bar
  • Encounter popup slides up from bottom
📖
Bestiadex
  • Grid of 30 creature cards (silhouette if not caught)
  • Filter by element (Fiamma / Adriatico / Bosco / Terra / Armonia)
  • Each card: name, element badge, rarity star
  • Tap → detail: lore, stats, duplicates count
  • 3 duplicates → Evoluzione button unlocks
  • Caught count badge (e.g. "18 / 30")
🎯
Missioni
  • Active story chapter (Rossini manuscript arc)
  • Current objective with progress bar
  • Side missions list (capture X, duel Y, scan QR)
  • Completed missions with rewards earned
  • Mystery/parchment visual register
  • Chapter unlock animation on completion
🎒
Zaino
  • Reti (catch tools) with quantity + type
  • Esche (lures) with remaining uses
  • Uova (eggs) with hatch progress
  • Battle items for duels
  • Mission items (physical + digital)
  • Shop button → buy more Reti with Oro
👤
Profilo + Leaderboard
  • Avatar (Google profile pic)
  • Level, total EXP, Bestie caught
  • Hall of Fame: past event badges
  • Best creatures showcase (top 3)
  • Live leaderboard (this session)
  • Top 10 by score
  • Your rank highlighted
  • Hall of Fame all-time rankings
