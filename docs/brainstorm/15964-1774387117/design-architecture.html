Key Architecture Decisions
Server-side RNG only — All encounter triggers, capture rolls, and duel damage are calculated on the server. Players cannot cheat by intercepting requests.
GPS + timer fallback — Browser Geolocation API, polling every 10s. If accuracy >50m or unavailable, timer-based encounters (60–180s random interval) keep the game running.
Single-tenant, event-scoped — One Supabase project, one admin team. Sessions are scoped: player progress, creatures, and leaderboard are per-session. Hall of Fame persists across seasons.
Google OAuth primary, magic link fallback — One-tap login on Android/iOS. GDPR-K checkbox for under-14s. Session JWT managed by Supabase Auth.
Offline partial via Service Worker — Inventory, Bestiadex, and active mission cached locally. Syncs on reconnect. Critical for spotty outdoor connectivity.
Cost: ~€0/month for MVP — Vercel Hobby (free), Supabase Free tier (500MB / 50k auth), Resend free (3k emails/month). Scale to paid only when needed.
