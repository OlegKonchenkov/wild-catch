3 Implementation Approaches

How should we structure and build WildCatch? Pick one — my recommendation is highlighted.

A

MVP-First — Ship fast, iterate

Build only Phase 1 (encounters, capture, Bestiadex, GPS, leaderboard) and run a real event ASAP. Add duels, missions, QR codes, shop in Phase 2 after real player feedback.

Pros

  • First event in 6–8 weeks
  • Real feedback before building complex features
  • Lower risk — validate concept early
  • GPS + capture loop is the core fun

Cons

  • No duels = less engagement
  • No missions = shallow narrative
  • May feel incomplete
Timeline: 6–8 weeks to first event
B

Full Phase 1+2 — Complete experience ⭐ Recommended

Build the complete game as designed in the GDD: encounters, capture, duels, missions, QR eggs, shop, admin dashboard, leaderboard — all before the first event. The escape room narrative requires missions and QR codes to work properly.

Pros

  • Complete escape room + creature experience
  • Missions give the story backbone
  • QR codes enable physical exploration
  • Admin can configure full events
  • Duels drive player-vs-player engagement

Cons

  • 10–14 weeks to first event
  • More upfront work
Timeline: 10–14 weeks to first full event
C

Parallel Tracks — Admin first, player second

Build the admin panel and creature/session management first, so you can start creating content (30 creatures, story, missions) while the player-facing app is being built in parallel. Requires 2 developers.

Pros

  • Content creation starts immediately
  • Admin team isn't blocked
  • Fastest overall delivery if 2 devs

Cons

  • Requires coordinating 2 parallel workstreams
  • API contracts must be agreed upfront
  • More complex project management
Timeline: 8–10 weeks with 2 devs
💡 Why I recommend B
The escape room + linear story design (A+B narrative mechanic) requires missions and QR codes to function — without them it's just GPS walking with creature encounters, not an escape room. The full Phase 1+2 build is the minimum viable event experience. The extra weeks are worth it to deliver the narrative layer that makes WildCatch different from simply a Pokémon GO clone.
