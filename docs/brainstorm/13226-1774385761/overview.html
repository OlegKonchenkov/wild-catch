WildCatch — GDD Analysis & Project Overview

Reading your Game Design Document — here's what we're building

Core Concept

An outdoor creature-catching PWA (mobile-first, no app store) inspired by Pokémon GO mechanics but legally independent — original IP, original creatures ("Bestie/Creature"), GPS-triggered encounters, duels, missions, and an admin panel for live event management.

Tech Stack (from GDD)

  • Next.js 14+ (App Router)
  • Supabase (DB + Auth + Realtime)
  • Vercel (hosting + CI/CD)
  • Resend (magic link email)
  • PWA (installable, offline partial)

Two Phases (from GDD)

  • Phase 1 MVP: Auth, GPS encounters, capture, Bestiadex, sessions, leaderboard
  • Phase 2: Duels (realtime), missions, eggs, QR codes, shop, admin dashboard
